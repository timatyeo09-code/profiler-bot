Option B · Vercel deployment (secure, for live use)

Your key lives on Vercel, hidden from users. This is what to use for the Skool classroom.

  1. Go to vercel.com and sign up (free).
  2. Create a new project. Upload this HTML file as index.html.
  3. Create a folder called api and add a file called chat.js — I'll send the code separately.
  4. In Vercel project settings → Environment Variables, add ANTHROPIC_API_KEY with your key as the value.
  5. Deploy. Drop the URL in your Skool classroom.

Once deployed, this page automatically detects the server endpoint at /api/chat and uses it — no further setup needed.
